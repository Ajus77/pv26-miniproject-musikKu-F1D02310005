# MusikKu — Music Practice Tracker

Aplikasi GUI berbasis PySide6 untuk mencatat dan melacak sesi latihan musik harianmu.

---

## Cara Menjalankan

### Jalankan aplikasi
```bash
python main.py
```

> Database `musikku.db` akan dibuat otomatis di folder `database/` saat pertama kali dijalankan.

---

## Teknologi
- **Python 3.10+**
- **PySide6** — GUI framework
- **SQLite** — Database lokal (via `sqlite3` bawaan Python)
